API Settings

Spaceship API
1
Login to Spaceship.com → Account → API
2
Click "Generate API Key" — copy your API Key & Secret
3
Paste below. Base URL: https://api.spaceship.com/v1/
Namecheap API
1
Login to Namecheap → Profile (top right) → Tools
2
Find API Access → Enable API → Copy your API Key
3
You also need to whitelist your IP in API settings (required!)
4
API endpoint: https://api.namecheap.com/xml.response
GoDaddy API
1
Go to developer.godaddy.com
2
Click "API Keys" → Create New API Key → Select Production
3
Copy both Key and Secret shown once after creation
4
API endpoint: https://api.godaddy.com/v1/domains
